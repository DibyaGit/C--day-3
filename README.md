# CSharp-Day-3

This repository contains C# projects developed as part of Day 3 of my C# learning journey. Each folder represents a distinct application built to reinforce core programming concepts and practical implementation.

## Projects Included

- **MediaPlayerApp**: A media player application demonstrating file handling, user interface design, and multimedia playback.
- **TextToSpeechApp**: Converts text input into spoken audio, showcasing integration with system speech APIs.
- **TimeZoneApp**: A utility for converting and displaying time across different global time zones.
- **UtilityApp**: A collection of helpful tools and functions for everyday coding tasks.

## Purpose

These projects are designed to solidify foundational C# skills, including object-oriented programming, .NET framework usage, and practical application development.

## Getting Started

1. Clone the repository.
2. Open the `.sln` file in Visual Studio or Visual Studio Code with .NET SDK installed.
3. Build and run any project to explore its functionality.
